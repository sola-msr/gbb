# gbb
Go Build Bat -GBB- For Windows (WIndows bat file)

# Usage

Open `go-build.bat`, rewrite the following location, save and use.

```bat
SET TARGET_DIR=

SET GOARCH=amd64
SET GOOS=linux

SET BUILD_OP=

SET TARGET_FILE_PATH=
```
